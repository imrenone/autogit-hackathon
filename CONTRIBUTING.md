# Contributing to autogit-hackathon

This is a hackathon. Read [HACKATHON.md](HACKATHON.md) for the full submission rules.

## Summary

- Add one `.md` file to the `templates/` folder
- Fill in the frontmatter (`title`, `app_type`, `wallet`)
- Open a PR
- If valid, your PR is auto-merged and 5 USDC is sent to your wallet automatically

## Rules

- One entry per GitHub account
- One PR per entry, one file per PR
- The `app_type` must be unique (not already in the repo)
- Your prompt must produce a working React app in AutoGit without manual edits
